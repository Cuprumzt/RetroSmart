# Support

RetroSmart is an open prototype. There is no guaranteed support SLA.

## Before Opening An Issue

Check:

- [README.md](./README.md)
- [Getting Started](./docs/Getting-Started.md)
- [Hardware Notes](./docs/Hardware-Notes.md)
- [Compatibility Matrix](./docs/Compatibility-Matrix.md)
- [Validation Checklist](./docs/Validation-Checklist.md)

## Good Issue Reports Include

- iPhone model and iOS version
- Xcode version when app build issues are involved
- ESP32 board profile and Arduino ESP32 version
- module type and firmware sketch
- wiring and power setup
- relevant serial logs or screenshots
- exact steps to reproduce

For module ideas, include the intended sensor or actuator, required pins, expected readings/actions, and whether you have matching hardware available.
